🔑 Twelve Data API Key Alma Rehberi

✅ Neden Twelve Data?

  • Ücretsiz Plan: Günde 800 request
  • Kapsamlı Veri: Hisse senetleri, forex, emtialar, indices
  • Güvenilir: Dünya çapında kullanılan profesyonel API
  • Kolay Entegrasyon: JSON formatında veri

📋 Adım 1: Twelve Data Hesabı Oluşturun

  1. Twelve Data'ya Git
  2. "Get Free API Key" butonuna tıklayın
  3. "Sign Up" ile hesap oluşturun
  4. Email adresinizi onaylayın
⚠️ Önemli: Geçerli bir email adresi kullanın, onay maili gelecek.

🔐 Adım 2: API Key'i Alın

  1. Hesabınıza giriş yapın
  2. "Dashboard" sayfasına gidin
  3. Sol menüde "API Keys" bölümünü bulun
  4. "Create New API Key" tıklayın
  5. Key'inizi kopyalayın (Örnek: a1b2c3d4e5f6...)
API Key Örneği: a1b2c3d4e5f67890abcdef1234567890

⚙️ Adım 3: GlobalBorsa'ya API Key Ekleyin

  1. config/api_keys.php dosyasını açın
  2. Bu satırı bulun:
define('TWELVE_DATA_API_KEY', 'demo');
  1. 'demo' yerine gerçek API key'inizi yazın:
define('TWELVE_DATA_API_KEY', 'a1b2c3d4e5f67890abcdef1234567890');
  1. Dosyayı kaydedin

🧪 Adım 4: Test Edin

  1. API Test Sayfasını açın
  2. "Live Mode" görmelisiniz
  3. AAPL hisse verisi başarıyla gelmeli
  4. Tüm kategoriler veri çekmeli

📊 Adım 5: Verileri Güncelleyin

API key çalıştıktan sonra:

  1. Ana Piyasalar Sayfası - Otomatik veri güncellemesi
  2. Landing Sayfası - Ticker'da gerçek fiyatlar

⚠️ Önemli Limitler

  • Ücretsiz Plan: 800 request/gün
  • Dakika başına: 8 request limit
  • Toplam kategoriler: 8 kategori x ~20 sembol = 160 request
  • Günde: 5 kez tam güncelleme yapabilirsiniz

🎯 API Key Başarıyla Kurulduktan Sonra:

  • ✅ Gerçek hisse fiyatları gelecek
  • ✅ Forex kurları güncel olacak
  • ✅ Emtia fiyatları canlı olacak
  • ✅ Index değerleri gerçek zamanlı
  • ✅ Otomatik 10 dakikada bir güncelleme

🚨 Sorun Giderme

❌ "API Key geçersiz" hatası:

  • Key'i doğru kopyaladığınızdan emin olun
  • Başında/sonunda boşluk olmasın
  • Tırnak işaretleri içinde olsun

❌ "429 Rate limit" hatası:

  • Günlük limit dolmuş, yarın tekrar deneyin
  • Çok sık güncelleme yapmayın

❌ "Veri gelmiyor" sorunu:

  • API test sayfasını kontrol edin
  • Internet bağlantınızı kontrol edin
  • Hosting provider API isteklerini engelliyor olabilir
